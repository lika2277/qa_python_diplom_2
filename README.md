## Дипломный проект. Задание 2: Тестирование API

### Автотесты для проверки API приложения, которое помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы тесты API, покрывающие методы создания, авторизации и обновления пользователя, а так же методы создания заказа и получения заказа

Процент покрытия 100% (отчет: `htmlcov/index.html`)

### Структура проекта

- `instances` - пакет, содержащий классы с методами взаимодействия для пользователя, заказа и ингредиентов
- `tests` - пакет, содержащий тесты, разделенные по классам. Например, `test_register.py`, `test_login.py` и т.д.
- `data` - пакет, содержащий пути для вызова запросов

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание отчета Allure**

>  `$ pytest  --alluredir=allure_results`

**Просмотр отчета Allure**

>  `$ allure serve allure_results`
